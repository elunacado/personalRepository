¿Cuál es la diferencia entre Internet y la World Wide Web (WWW)?

Internet es una red enorme de computadoras conectadas entre ellas alrededor del mundo Mientras que la WWW es una enorme colección de páginas que se encuentran sobre el internet, por lo que necesitas de este último para acceder a la web.

Las partes que componen un URL son: https://es.wikipedia.org/wiki/Hockey_sobre_hielo

  • Subdominio: www
  • Dominio: Wikipedia

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

  • GET: Se encarga de la recuperación y representación de datos
  • HEAD: pide una respuesta idéntica a la de una petición GET, pero solo el encabezado de esta petición
  • POST: Enviar datos para su procesamient
  • PUT: Actualiza un recurso
  • PATCH: Modifica solo una parte de los datos de un recurso
  • DELETE: Borra un recurso en específico

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

El error 404 significa File Not Found y esto se puede deber tanto a que el usuario buscó el file equivocado como que el desarrollador no coloco respuesta para esta petición por algún error interno.

Sobre el ciclo de vida y desarrollo de los sistemas de información: ¿Cuál es el ciclo de vida de los sistemas de información?

  • El sistema nace al encontrarse una problemática y decidir solucionarla
  • El sistema se desarrolla cuando este se crea como tal
  • El sistema se reproduce cuando este es puesto a prueba y se corrigen sus errores
  • El sistema muere cuando este queda obsoleto o deja de funcionar con la misma efectividad
